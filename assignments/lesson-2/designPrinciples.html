Design Principles

Proximity

Roolee
https://roolee.com/

This website has great proximity between the various elements (text, images and icons etc.). The even sized colored images break up the white background where the type is. The website also has great negative space, which enhances the proximity. For example, the negative space between the images/icons and the text help the viewer group the three lines of text together and the images (which include text) separately.

Alignment

Studio McGee
https://www.studio-mcgee.com/

The Studio McGee mobile website is simple, but uses very effective alignment, which is also supported by great repetition. The logo, text and images are all center aligned down the page. The Studio McGee website, like the Roolee website also uses great negative space, which makes the alignment and repetition even more apparent.

Repetition

Weight Watchers
https://www.weightwatchers.com/us/

The Weight Watchers website uses great repetition in a variety of ways. The first examples of repetition are the white boxes which are displaying the program options. They are obviously both white rectangles with images cropped to circles, bold titles, and blue hyperlinks, which are both left aligned. The text is all the same typeface as well and there is a cohesive color scheme which helps the flow of the website.
